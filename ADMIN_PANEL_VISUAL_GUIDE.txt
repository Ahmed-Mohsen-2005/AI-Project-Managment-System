9. DEFAULT ROLES & THEIR PURPOSES
════════════════════════════════════════════════════════════════════════════════

ADMIN
└─ Full system access with administrative privileges
   ├─ Can create/edit/delete everything
   ├─ Can access all data
   └─ Can manage roles and permissions

PROJECT MANAGER
└─ Can manage projects, tasks, and team members
   ├─ Access: Sprint Boards, Reports
   ├─ Can create and manage projects
   └─ Can assign tasks to team members

TEAM MEMBER
└─ Can view and edit assigned tasks and projects
   ├─ Access: Sprint Boards, Reports
   ├─ Can update their own tasks
   └─ Limited to assigned projects

CLIENT
└─ Limited view-only access to project information
   ├─ Access: Reports (limited)
   ├─ Can only view project status
   └─ Cannot create or modify anything

INTERN
└─ Restricted access with limited editing privileges
   ├─ Access: None by default
   ├─ Can be granted limited permissions
   └─ Usually for training/observation

10. KEYBOARD SHORTCUTS & TIPS
════════════════════════════════════════════════════════════════════════════════

Tab Navigation:
  • Click any tab in the left sidebar
  • Tabs show: Roles, Permissions, Access Control, Users, Audit Log

Search:
  • Use search bar at the top to filter current tab
  • Real-time filtering
  • Works on roles, permissions, and users

Forms:
  • All required fields marked with *
  • Click Edit to open form with existing data
  • Click Save to update
  • Click Cancel to close without saving

Matrix:
  • Click checkboxes to toggle access
  • Changes save instantly
  • No save button needed

11. COMMON TASKS QUICK GUIDE
════════════════════════════════════════════════════════════════════════════════

Task: Create a new role
  1. Go to "Roles Management" tab
  2. Click "+ Add Role" button
  3. Enter role name and description
  4. Click "Save"

Task: Restrict access to Sprint Boards
  1. Go to "Access Control" tab
  2. Find the role row
  3. Uncheck "Sprint Boards" checkbox
  4. Done! (saves automatically)

Task: Assign a role to a user
  1. Go to "User Roles" tab
  2. Find the user
  3. Select role from dropdown
  4. Done! (saves automatically)

Task: View admin action history
  1. Go to "Audit Log" tab
  2. Optionally filter by date or action
  3. Scroll through all logged actions

Task: Add a new permission
  1. Go to "Permissions Management" tab
  2. Click "+ Add Permission" button
  3. Enter name and description
  4. Click "Save"
